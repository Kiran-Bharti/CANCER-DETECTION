STEP 1: GO TO THE DETECT PAGE


1. Click on the Choose button.

2. Select the file/folder.

* The images that are being uploaded should be Histopathologic images.

* The file should be either in .jpg, .jpeg or .tif format only.

* If the file is in any other format or there is no file selected then a warning will be shown.

3. After selecting the file/folder click on the UPLOAD button.

STEP 2: VIEW THE RESULT


As soon as you have uploaded your file/folder our model will start working on it.

The detection may take some time so you might have to wait for some moments.

After the detection process is complete you can view the results.


The results have 80% accuracy. If the result is positive for cancer then it means that there are 80 percent chances that cells have cancer. In such a case medical actions should be taken immediately.

For a negative result there are still 20 percent chances of cancer. So, in that case some more samples should be uploaded for detection. If the results remain same after 2-3 attempts, then the cells are free from cancer.
